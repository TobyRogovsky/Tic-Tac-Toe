# Tic-Tac-Toe

## Overview
this repo is for the implementation of the classic game of Tic Tac Toe. thie software will allow two players to play against each other or one player to play against the computer.
a description of the game plus the requirements of this implementation is provided below.

## The Game
Two player are presented with a 3x3 grid of empty squares. the players take turns marking an empty spot with their letter: the first player to move is X and the second player is O. the first player to fill a winning set, defined as a row, column, or diagonal line of 3 squares, wins. if all spots are filled and neither player has a winning set, the game results in a tie.
